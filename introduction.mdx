![Partoo Rest API Logo](https://partoo-client-images.s3.amazonaws.com/logo-partoo-restapi-white.png)

The Partoo Rest API allows you to automate all the actions that are possible to do in the Partoo Web Application. Use it to create, update, or delete businesses and users, manage organizations (if you're a reseller), retrieve data, and much more.
